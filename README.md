Dengan menggunakan pointer, kita dapat mengakses atau memanipulasi nilai variabel yang terletak di lokasi memori tertentu. Pointer untuk Mengakses Array digunakan untuk mengakses elemen-elemen dalam array dengan memanfaatkan konsep alamat memori. Misalnya, pointer dapat diarahkan ke alamat pertama dalam array, dan dengan menggunakan pointer aritmetika, kita dapat mengakses elemen-elemen array berikutnya.

# Jobsheet 10
Modul ini berharap kita mampu menjelaskan dan mengimplementasikan Enum dan Structure dalam pemrograman menggunakan IDE. Enum sendiri adalah tipe data yang menyusun kumpulan konstanta bernilai tetap dengan memberikan nama simbolik pada setiap nilai. Ini memudahkan pembacaan dan pemeliharaan kode. Struct adalah tipe data yang memungkinkan penggabungan berbagai jenis data, seperti integer, float, dan char, dalam satu entitas. Struktur ini memungkinkan pembuatan tipe data khusus dengan properti yang berbeda. Enum digunakan untuk membuat kumpulan konstanta bernama, memberikan nilai yang jelas dan mudah dimengerti. Struct digunakan untuk menggabungkan beberapa jenis data menjadi satu entitas. Misalnya, struct dapat digunakan untuk mewakili data mahasiswa dengan nama, usia, dan nilai sebagai propertinya. Typedef pada struct digunakan untuk memberikan nama alias yang lebih singkat atau mudah diingat untuk tipe data struct tertentu. Ini membantu meningkatkan kejelasan dan keterbacaan kode. Struct bersarang adalah penggunaan struktur di dalam struktur lainnya. Ini memungkinkan pembuatan hierarki data yang kompleks dengan tingkat kedalaman yang berbeda. Contoh: struct yang mewakili data perusahaan dengan anggota struct karyawan di dalamnya. Struct dapat dilewatkan ke dalam fungsi sebagai parameter, memungkinkan manipulasi atau pengolahan data di dalam fungsi. Ini dapat dilakukan dengan menyertakan struct sebagai argumen fungsi atau menggunakan pointer pada struct tersebut.
